I've got a folder of scattered notes — half-finished thoughts on things I'm building, bugs I chased down, ideas I didn't want to lose. This is where they go now.

No pressure for these to be polished essays. Some entries will be three sentences. Some will have code in them. The point is just to keep a record, in public, of what I'm figuring out.

To add a new one: drop a markdown file in `_posts/` named `YYYY-MM-DD-some-title.md`, like this one, with a title and optional tags at the top. That's it — it shows up on the log automatically, newest first.
